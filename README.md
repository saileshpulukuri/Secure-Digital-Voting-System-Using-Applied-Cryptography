# Secure Digital Voting System Using Applied Cryptography

A secure digital voting system that leverages applied cryptography to ensure voter privacy, ballot integrity, and verifiable election outcomes.

## Overview

This project implements a cryptographically secure digital voting platform. The system is designed to provide:

- **Privacy** — Voters can cast ballots without revealing their choices to anyone
- **Integrity** — Ballots cannot be altered or forged after submission
- **Verifiability** — Voters can verify their vote was counted; the public can verify the tally
- **Transparency** — Election process and results can be audited without compromising privacy
